Introduction to Node.js

"Introduction"

So, let us begin with what is node.js and its applications. Node.js is a platform built on Chrome’s JavaScript runtime for easily building fast and scalable network applications. It is a light, scalable, and cross -platform way to execute code. It is primarily used for non -blocking, event driven servers, due to its single threaded nature. It is used for building traditional websites and back-end API services. Node.js is not a programming language, but it allows developers to use JavaScript which allows users to build web applications
