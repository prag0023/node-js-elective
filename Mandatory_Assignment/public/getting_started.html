"Getting started"

Moving onto VS code, right after opening the VS code you will get a welcome text which you just cancel then as you have already set up your directory. You can right click and create your new folder give the name that you want. In my case, I created a folder named Node.js and inside that folder create a file test.js. Inside the file we write codes that we want to execute. For example, console.log (“hey I am writing some documentation”). We run our code through terminal, so if you look up in the screen you can see the terminal open it and go into the current working directory which you have set up before. Then node (file name) example:node test.js It will run your code and you should be able to see the output. So, its basically how you get started with the VS code and run your first test.js file. To share and save your codes, create a GitHub account and create a repository. To push your code to GitHub, you go to vs terminal and
Git add -A: which adds your files.
Git commit -m “Type your message”: about what you made changes in the code
Git push: pushes your code into your repository.

Tools and techniques used
Nodemon: Nodemon is a tool that helps develop node. js based applications by automatically restarting the node application when file changes in the directory are detected. Nodemon is a replacement wrapper for node .
Express: It's a web framework that let's you structure a web application to handle multiple different http requests at a specific url. Express is a minimal, open source and flexible Node. js web app framework designed to make developing websites, web apps, & API's much easier.
Vs code: Visual Studio Code is a code editor redefined and optimized for building and debugging modern web and cloud applications.
package.json: A package.json is a JSON file that exists at the root of a Javascript/Node project. It holds metadata relevant to the project and it is used for managing the project's dependencies, scripts, version etc.
npm: Npm is the default package manager for the JavaScript runtime environment Node.js which provides hosting for software development and version control with the usage of Git.
