Creating a rest API project having CRUD functionality on VS code with node.js

CRUD is an acronym that comes from the world of computer programming and refers to the four functions that are considered necessary to implement a persistent storage application: create, read, update and delete.
Here, we have created a project applying the all the four functionality of CRUD with hardcoded database and serving it with express. So, lets get started.
First create a folder in Vs then change your directory into the folder. Inside the folder create a file (package.json) Inside the package.json file give the dependencies and the version of express


Then, we move to our Vs terminal where we install express with the command : npm install express


When we install express it creates package-lock-json and pulls all the node modules. Now we move forward and create app.js which we be the main file which will contain our all routes and main running file. In the app.js file

Here, we have first initialized our express app and the gave the port where our server can run.

POST method



Post method is used for creating resource. You need to have a response body in the post method. You cannot send post, patch, or delete request in the url directly it is only for the get request. So we are using a tool called post man to test our all http methods.


PATCH METHOD



It updates a part of resource meaning it is used for making partial changes to an existing resource. The PATCH method provides an entity containing a list of changes to be applied to the resource requested using the HTTP Uniform Resource Identifier(URI).
br>
DELETE METHOD



It deletes the resource. The DELETE method requests that the origin server delete the resource identified by the request-URI.
